OTP Generation and Verification Over Cloud Environment.

Platform Used: Amazon Web Services (AWS).

The moto of developing otp generation and verification on a cloud environment is to overcome the limitations
of traditional on-premise otp systems and provide a secure, scalable,cost-effective, and highly available
authentication mechanism for modern applications. Traditional systems struggle with issues like slow otp
delivery, high infrastructure cost,security vulnerabilities, and low scalability during high traffic. A cloud
based solution using services like api gateway, lambda,dynamodb, and amazon ses eliminates these problems by
offering serverless architecture,automatic scaling and strong security controls. This ensures that otps are
generated,stored, and verified securely and efficiently without manual intervention.cloud systems also provide
automatic otp expiration using ttl,faster email delivery through ses,built-in monitoring/logging,and reliable
performance even during peak loads.

Online Examination Portal.

Technologies Used: HTML,CSS,JavaScript,JDBC,Servlets,JSP,MY SQL.

Software Used: Eclipse IDE.

The moto of developing online examination portal is developed to overcome the limitations of traditional paper
based exams by offering a flexible, secure, and efficient system that allows students to take tests from any -
where while reducing manual work and administrative costs.It ensures instant evaluation, easy management of
questions, and a seamless exam experience. The portal uses html for structuring pages, css for styling and
responsiveness,and JavaScript for dynamic behavior like timers and form validation. mysql stores user details,
questions, and results, while jdbc enables smooth database connectivity. servlets handle backend logic such as
user authentication, question loading, and result processing to deliver a reliable and scalable examination system.

IoT Based Smart Parking System.

Software Used: Blynk IoT.

Hardware Components Used: NodeMCU ESP8266,Ultrasonic Sensors.

An IoT-based smart parking system is needed to reduce traffic congestion, save driver's time, and avoid fuel
wastage caused by searching for free parking slots in busy areas. This system helps users find available slotsin
real time and improves overall parking efficiency. It uses ultrasonic sensors to detect vehicle presence, a
NodeMCU ESP8266 to collect and send data, and the Blynk IoT platform to display slot status on a mobile app.
The system updates availability instantly, enabling smart monitoring, automated slot allocation, and a smoother
parking experience in crowded locations.
